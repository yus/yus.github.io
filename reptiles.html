Escher-Style Lizard Tessellation

Inspired by M.C. Escher's Reptiles (1943)

This SVG illustration recreates the iconic lizard tessellation style made famous by M.C. Escher. The design features interlocking lizards that fit together perfectly without gaps, demonstrating Escher's mastery of mathematical art.

The central lizard is highlighted with more detail, while the surrounding lizards create a tessellating pattern. Each lizard is constructed using SVG path elements with curved segments that interlock perfectly.
