About This Animation

This demonstrates 3D rotation of an SVG element using CSS transforms. The mathematical symbol (U+2A33) rotates independently around the X, Y, and Z axes.

Use the sliders to manually control rotation on each axis, or start the auto-rotation to see a continuous 3D rotation loop.

The SVG has been centered in its container using the original transform values, and the 3D rotation is applied around the center of the symbol.
