Conway's Game of Life with Floating States

Click on cells to toggle them between states. Each click changes the color palette for new cells.

10 FPS

How it works: This is a modified Conway's Game of Life with 5 states (0, 0.25, 0.5, 0.75, 1). Each state is represented by a color from the current palette. Clicking on a cell cycles its state and changes the palette for new cells.

Current Palette: Sunset

Rules: Cells with 2-3 live neighbors survive, cells with exactly 3 neighbors become alive, others die. "Live" means state > 0.
